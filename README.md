# 數學小樂園 🐱🐶

幫一年級小朋友學習 **18以內加減法** 的互動式網頁遊戲。用動畫一步步拆解湊10法和拆10法，邊玩邊學。

🔗 **線上玩：** `https://你的username.github.io/math-games`

---

## 遊戲內容

### 🐱 貓咪加法樂園 (`cat-math.html`)
- 18以內加法
- 核心策略：**湊10法**（ex: 6+5 → 拆成 6+4+1 → 10+1=11）
- 十格小屋視覺化，幫助理解數量關係

### 🐶 狗狗減法樂園 (`dog-math.html`)
- 18以內減法
- 核心策略：**拆10法**（ex: 15-8 → 拆成 10+5-8 → 10-8+5=7）
- 支援三種題型：需借位、不需借位、純個位

---

## 功能特色

- **動畫教學**：步驟分解動畫，看懂才做題
- **自己算算看**：四選一答題，附視覺化提示
- **防重複題目**：每次練習題目不重複
- **五個一排**：符合五進位直覺，一眼看出數量
- **手機友善**：支援手機與電腦

## 教學設計說明

| 題型 | 策略 | 步驟 |
|------|------|------|
| 6 + 7 | 湊10法 | 6+4=10，10+3=13 |
| 15 - 8 | 拆10法（需借位） | 15=10+5，10-8=2，2+5=7 |
| 14 - 3 | 拆10法（不借位） | 14=10+4，4-3=1，10+1=11 |
| 9 - 4 | 直接減 | 9-4=5 |

設計原則參考：認知負荷理論（工作記憶限制）、具體→圖像→抽象的學習順序。
